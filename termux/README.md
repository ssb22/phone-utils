# Utilities for Termux
* [Gemini reader](gemini-reader.py) uses Google's Gemini TTS
  service to read the clipboard
* [VBR MP3 coder](mp3-vbr.sh) recodes "Amazing MP3 Recorder"
  OGG files to VBR MP3, places in Downloads and initiates
  Share (for sending longer voice notes to a chat app like
  Discord more robustly)
